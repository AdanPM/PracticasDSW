HTML son las siglas de Hypertext Markup Languaje, que puede traducirse como lenguaje de marcas o marcado de hipertexto

El lenguaje HTML se emplea para crear las páginas web. Es muy fácil ver eel código HTML de una página web, la opción exacta cambia de un navegador a otro y también puede cambiar de una versión a otra de un mismo navegador, pero suelen tener un nombre similar.

HTML se compone de etiquetas que se escriben entre los simnolos menor que y mayor que.

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creó la primera página web
